## MRP System

Mobile Reimbursement Portal built with React + Vite + Supabase.

## Setup
1. Clone the repo
2. Follow SUPABASE_SETUP.md to set up the backend
3. Add .env file with your Supabase keys
4. npm install
5. npm run dev

This project uses Supabase as the backend.
See SUPABASE_SETUP.md to replicate the backend structure.
To use a different backend, replace all supabase.* calls
with your own API calls.
